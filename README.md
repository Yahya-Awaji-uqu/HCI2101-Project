## 📌 Project Overview
This is a responsive, static course website built for **Human-Computer Interaction Fundamentals (HCI 1201)**. The project demonstrates core UI/UX principles, strictly adhering to the requirement of using **pure HTML and CSS only** (No JavaScript, No external frameworks like Bootstrap).

### 🚀 Live Demo
**[Click here to view the live website](https://yahya-awaji-uqu.github.io/HCI2101-Project/)**

## ✨ Key Features
- **Bilingual Interface:** Full support for both English (LTR) and Arabic (RTL) reading directions.
- **Dual Theme:** Seamless Light and Dark modes integrated using CSS Variables (`:root` and `.dark-mode`).
- **Responsive Design:** Fully adaptive layouts for Desktop, Tablet, and Mobile screens using CSS Flexbox and Grid.
- **Clean Architecture:** Semantic HTML structure and a single, well-organized `style.css` file.
